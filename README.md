# 📚 Bookstore Management System

A Java Swing application connected to MySQL using JDBC.

## Features
- Add Book
- Search Book
- Sell Book
- Update Stock
- View Inventory

## Technologies Used
- Java
- Swing
- MySQL
- JDBC

## How to Run
1. Create MySQL database named `bookstore`
2. Create `books` table
3. Add MySQL connector JAR
4. Run Bookmanagement.java
